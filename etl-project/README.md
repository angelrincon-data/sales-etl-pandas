\f0\fs24 \cf0 # Sales ETL Pipeline with Python & Pandas\
\
## \uc0\u55357 \u56524  Project Overview\
This project implements an end-to-end ETL (Extract, Transform, Load) pipeline using Python and Pandas.\
\
The pipeline processes raw sales data, cleans and transforms it, and produces an analytical dataset ready for reporting or BI tools.\
\
## \uc0\u55357 \u56770  Project Structure\
\
\
## \uc0\u55357 \u56580  ETL Steps\
### Extract\
- Load CSV files (customers, products, sales)\
\
### Transform\
- Data type normalization\
- Date parsing\
- Missing values handling\
- Referential integrity checks\
- Business metrics calculation\
\
### Load\
- Export final analytical dataset to CSV\
\
## \uc0\u55357 \u57056  Tech Stack\
- Python\
- Pandas\
- Jupyter Notebook\
\
## \uc0\u55357 \u56522  Output\
- `sales_analytics.csv` ready for analysis or dashboards\
